# Protein Physicochemical Characterization Using Biopython

Developed a Python-based bioinformatics workflow to retrieve protein sequences from UniProt and perform physicochemical characterization. The pipeline calculates key protein properties including molecular weight, theoretical isoelectric point (pI), and amino acid composition using Biopython, demonstrating fundamental skills in sequence analysis and computational biology.

Protein sequence analysis
Computational biology
Bioinformatics pipelines
Python programming
Biopython
Data handling and reporting
